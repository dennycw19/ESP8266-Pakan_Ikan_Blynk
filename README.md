# ESP8266-Pakan_Ikan_Blynk
Automatic fish feeder using ESP8266 integrated with Blynk

The components used in this tool are:
1. NodeMCU V3 ESP8266
2. NodeMCU V3 ESP8266 Base Plate Board
3. LCD 16x2 I2C
4. Servo MG995
5. RTC 3231
6. DS18B20
7. Resistor 4.7K Ohm
